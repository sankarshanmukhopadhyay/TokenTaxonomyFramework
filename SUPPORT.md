# Support

This repository is a public artifact and documentation surface for the Token Taxonomy Framework.

## Questions and discussions

Use GitHub issues or the appropriate InterWork Alliance community channel for questions about:

- taxonomy concepts;
- artifact format and repository structure;
- documentation gaps;
- dMRV samples and extension sets;
- validation failures;
- contribution process.

## Before requesting support

Please include:

- the file or artifact path involved;
- the expected behavior or interpretation;
- the actual issue observed;
- any validation output from the scripts in `scripts/`;
- whether the request concerns documentation, JSON artifacts, proto definitions, or dMRV extension packages.
